# Grid Source-ordered Hero  
  
Using Grid opens up some unique source order options.  
  
```  
+-------------+  
|             |  
|  a          |  
|             |  
+-------------+  
|             |  
|  b (image)  |  
|             |  
+-------------+  
|             |  
|  c          |  
|             |  
+-------------+  
  
  
+-----------+--------+  
|           |        |  
|  a        |  b     |  
|           |        |  
+-----------+        |  
|           |        |  
|  c        |        |  
|           |        |  
+-----------+--------+  
```
